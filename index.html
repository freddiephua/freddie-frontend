How to add or swap books
1. Open Claude (claude.ai) and find the Freddie conversation
2. Upload your book PDF or epub and say "add this to my Reading tab"
3. Claude extracts key concepts tailored to you
4. You receive a new index.html file
5. Go to github.com/freddiephua/freddie-frontend
6. Click index.html → pencil icon → select all → paste → commit
7. Vercel redeploys in ~30 seconds
